# C++ Time Formatting and Menu System

## Overview

This C++ program displays time in both 12-hour and 24-hour formats and allows the user to interact through a simple menu. It updates time and keeps both formats synchronized.

## Problem Being Solved

The program converts and displays time accurately between 12-hour and 24-hour formats while providing an easy-to-use interface.

## What I Did Well

I created a clean menu loop, handled time conversion correctly, and kept the output consistent and easy to read.

## Areas for Improvement

I could improve input validation, break the code into more functions, and add features like setting custom time.

## Challenges and Solutions

Handling AM/PM transitions and avoiding infinite loops were challenging. I solved these by testing edge cases and refining the logic.

## Transferable Skills

This project strengthened my skills in loops, conditionals, debugging, and writing clean, readable code.

## Maintainability

The code is organized, readable, and structured in a way that makes it easy to update and expand later.
